# F1 Analysis Project

This project analyzes Formula 1 race data using FastF1 and Python. It includes lap time visualizations, driver performance comparisons, and strategy insights.

## Features

- Lap time distributions and scatter plots
- Grid vs. finish position analysis
- Tyre compound and life comparisons
- Driver and team performance metrics

## Setup

1. Install dependencies: `pip install fastf1 matplotlib seaborn pandas`
2. Load data: Run the notebook cells to fetch F1 session data.
3. Analyze: Execute cells for visualizations and tables.

## Usage

- Open `Driver_notebook.ipynb` in Jupyter.
- Run cells sequentially to load data and generate plots.
- Customize drivers, sessions, or compounds as needed.

## Data Source

Data is fetched from FastF1 API for official F1 sessions.
